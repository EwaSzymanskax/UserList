# UsersList
> The app allows to add a specific number of users and displays them as a list.
> Live demo [_here_](https://ewaszymanskax.github.io/UsersList/)

## Table of Contents
* [General Info](#general-information)
* [Technologies Used](#technologies-used)
* [Setup](#setup)
* [Acknowledgements](#acknowledgements)


## General Information
- The users list consists of: username and age. 
- The max number of users is defined by the application.


## Technologies Used
- ReactJS
- CSS modules


## Setup
- download or clone the repository
- run: npm install
- start server: npm start
